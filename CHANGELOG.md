# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-10-16

### Added
- Initial release
- 9 read-only tools for Exceptionless API
- 6 Event tools: get-events, get-event, get-event-by-reference, count-events, get-sessions, get-session-events
- 3 Stack tools: get-stacks, get-stack, get-stack-events
- Token-optimized responses (summary mode, compact JSON)
- Production-ready error handling and retry logic
- Comprehensive documentation
- Unit and integration tests
